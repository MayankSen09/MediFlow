# 🏥 MediFLow-Hospital Management System

MediFlow centralizes all hospital workflows into a single platform including patient management, appointments, billing, pharmacy, laboratory, and staff operations with a focus on security, performance, and usability.

## ✨ Features

### 🔐 **Authentication & Security**
- JWT-based authentication
- Role-based access control
- Secure login system

### 👥 **Patient Management**
- Patient registration and profiles
- Medical history tracking
- Search and filtering
- Admission/discharge workflow

### 📅 **Appointment System**
- Interactive scheduling
- Doctor availability management
- Status tracking (Scheduled, Confirmed, Completed)
- Conflict detection

### 👨‍⚕️ **Doctor Management**
- Doctor profiles and specializations
- Performance tracking
- Schedule management

### 🏨 **Ward & Bed Management**
- Real-time bed occupancy
- Visual ward layouts
- Patient admission/discharge
- Occupancy analytics

### 💊 **Pharmacy Management**
- Medicine inventory tracking
- Prescription processing
- Stock level alerts
- Expiry date management

### 🔬 **Laboratory Management**
- Test ordering system
- Result management
- Priority handling (Normal, Urgent, STAT)
- Turnaround time tracking

### 💰 **Billing & Payments**
- Invoice generation
- Multiple payment methods
- Revenue tracking
- Payment history

### 👥 **Staff Management**
- Employee directory
- Attendance tracking
- Shift scheduling
- Payroll management

### 📊 **Reports & Analytics**
- 8+ pre-built report templates
- Real-time data visualization
- Export to PDF, Excel, CSV
- Scheduled reports
- Custom filtering

## 🔧 **Installation & Setup**

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/hospital-management-system.git
   cd hospital-management-system
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🌐 **Deployment**

### **Vercel (Recommended)**
1. Connect your GitHub repository to Vercel
2. Configure build settings:
   - Build Command: `npm run build`
   - Output Directory: `build`
3. Deploy automatically on push

### **Other Platforms**
- **Netlify:** Drag and drop the `build` folder
- **GitHub Pages:** Use `gh-pages` package
- **AWS S3:** Upload build files to S3 bucket
